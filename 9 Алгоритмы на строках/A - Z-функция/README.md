# A. Z-функция
Ограничение по времени на тест : 1 секунда

Ограничение по памяти на тест : 256 мегабайт

#

### Выведите z−функцию для заданной строки.

#

## Входные данные
В первой строке дана строка S (1 ≤ | S | ≤ 10<sup><i>5</i></sup>) состоящая из маленьких латинских букв.

## Выходные данные
В единственной строке выведите | S | чисел через пробел — значения z−функции для заданной строки.

#

## Пример

### Входные данные :
abacaba
### Выходные данные :
7 0 1 0 3 0 1
